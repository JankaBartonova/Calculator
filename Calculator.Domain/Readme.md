# Calculator

Usage: dotnet Calculator.Domain.dll [command]

Calculates basic operatons of basic shapes.
Calculator executable via command line 

Shapes:
- square
- rectable
- cube
- block

Operations:
- Area
- Circumference
- Surface
- Volume

Commands:
- SquareArea <a>
- SquareCircumference <a> <b>
- RectangleArea <a> <b>
- RectableCircumference <a> <b>
- CubeSurface <a>
- CubeVolume <a>
- BlockSurface <a> <b> <c>
- BlockVolume <a> <b> <c>
